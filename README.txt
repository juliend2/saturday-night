The WordPress plugin for party animals.

Surround any text you want with these shortcodes: [party], [rainbown], [fire], [shit], [boom].

== Installation ==
1. put the folder into the /wp-admin/plugins/ directory
2. Activate the plugin
3. Open a beer
4. Party!!!
